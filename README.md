# Housing Dataset - Exploratory Data Analysis (EDA)

## 📌 Objective
To analyze the housing dataset using statistical methods and visualizations to identify patterns, trends, and relationships between features affecting house prices.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Steps Performed
- Loaded dataset using Pandas
- Generated summary statistics (mean, median, std)
- Checked missing values
- Created histograms for data distribution
- Used boxplots to detect outliers
- Built correlation heatmap to understand relationships
- Used pairplot for feature comparisons

## 🔍 Observations
- Price increases with increase in area
- Bedrooms and bathrooms positively influence price
- Area has strong correlation with price
- Some outliers are present in price and area
- Data shows slight skewness in distribution

## ✅ Conclusion
EDA helped in understanding key factors influencing housing prices and identifying patterns useful for further machine learning models.
